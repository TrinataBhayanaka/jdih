Menampilkan Daftar Produk Hukum Berdasarkan filter

No Parameter Keterangan Contoh URL
1. page Menampilkan list data (maksimal 50 record) page=1
(menampilkan record ke 1 - 50) page=2
(menampilkan record ke 51 - 100)
dst 		http://localhost/jdih/services/home/listData/?page=1
2. first_date Menampilkan data berdasarkan tanggal (format : yyyy-mm-dd) first_date=2016-07-27
(menampilkan data dengan tanggal >=2016-07-27)
 http://localhost/jdih/services/home/searchData/?page=1&first_date=2016-07-27
3. last_date Menampilkan data berdasarkan tanggal (format : yyyy-mm-dd) first_date=2016-07-27
(menampilkan data dengan tanggal <=2016-07-27)
 http://localhost/jdih/services/home/searchData/?page=1&last_date=2016-07-27
4. produk Menampilkan data berdasarkan jenis produk hukum
*)referensi jenis produk hukum		 produk=1
(menampilkan data dengan jenis produk hukum UUD Tahun 1945)
http://localhost/jdih/services/home/searchData/?page=1&produk=1
5. status Menampilkan data berdasarkan status jenis produk hukum (Rancangan,Berlaku,Perubahan,Pencabutan) status=Rancangan
(menampilkan data jenis produk hukum dengan status rancangan)
 http://localhost/jdih/services/home/searchData/?page=1&status=Rancangan
